# Decodelabs-Artificial-Intelligence-Project-4
# 🤖 AI Text Recognition System (OCR)

A beginner-friendly Artificial Intelligence project that extracts text from images using Optical Character Recognition (OCR).

This project was developed as part of the DecodeLabs Artificial Intelligence Internship Program.

---

# 📌 Project Overview

The AI Text Recognition System uses a pre-trained OCR engine to identify and extract text from image files.

Instead of training a model from scratch, this project demonstrates how AI engineers can integrate existing AI libraries into practical applications.

The system accepts an image as input, processes it using OCR technology, and displays the extracted text.

# ⚙️ Installation

Install required libraries:

pip install pillow pytesseract

Install Tesseract OCR Engine:

https://github.com/tesseract-ocr/tesseract

Make sure Tesseract is installed and added to your system PATH.

---

# ▶️ How to Run

Run:

python main.py

Enter the image path when prompted.

Example:

sample_image.jpg

The system will extract text and save the result to:

recognized_text.txt
